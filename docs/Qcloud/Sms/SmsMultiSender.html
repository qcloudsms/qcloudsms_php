普通群发

普通群发需明确指定内容，如果有多个签名，请在内容中以【】的方式添加到信息内容中， 否则系统将使用默认签名。

Parameters

int $type 短信类型，0 为普通短信，1 营销短信
string $nationCode 国家码，如 86 为中国
array $phoneNumbers 不带国家码的手机号列表
string $msg 信息内容，必须与申请的模板格式一致，否则将返回错误
string $extend 扩展码，可填空串
string $ext 服务端原样返回的参数，可填空串
